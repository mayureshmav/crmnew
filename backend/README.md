# CRM Backend (Spring Boot)

This is the backend structure for the CRM automation project. Modules:
- Lead & Opportunity Management
- Client Intake & Discovery
- Costing & Quotation
- Project & Task Management
- Layouts & Designs
- Procurement & Vendor Management
- Invoice Management
- Project Tracking

## Structure
- src/main/java/com/crm/lead
- src/main/java/com/crm/clientintake
- src/main/java/com/crm/costing
- src/main/java/com/crm/project
- src/main/java/com/crm/layouts
- src/main/java/com/crm/procurement
- src/main/java/com/crm/invoice
- src/main/java/com/crm/tracking

You can now add controllers, services, repositories, and models in each module folder as needed.
